How to use

When there is no usage for 15 minutes, the server shuts down. When you load the page, the server will begin booting up again. It can take up to a minute for you to be able to chat.

The public chat room is called Public. Join it to chat with others. For private chats, host any Room ID and share it.

To create a custom room ID, type it in and click Host Room. Share your room ID with others to connect.
