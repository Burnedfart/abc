How to use

The public chat room is called RoomOne. If you want to chat with a bunch of other people, join RoomOne, it's always being hosted. If you click host on an ID that is already hosted, you'll just be connected. If you want to have a private chat with a friend, host any Room ID and make sure your friend can get it (email, predetermined, etc).

If you're using a VPN, please turn it off before using, as it will cause errors and random disconnections otherwise.

To create a custom room id, type it in and click Host Room.

Share your room ID and have the other person enter it in the Connect field, then click Connect.
